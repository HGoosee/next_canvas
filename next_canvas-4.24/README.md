# Next Canvas - 轻量级绘图应用

Next Canvas 是一个基于 React 和 Konva.js 构建的轻量级绘图应用，提供直观的用户界面和丰富的绘图功能。

## 功能特点

- **多种绘图工具**：支持矩形、圆形、三角形、星形等多种形状
- **文本编辑**：添加和编辑文本内容
- **图片导入**：支持导入和调整图片
- **画布操作**：缩放、平移和选择功能
- **历史记录**：支持撤销和重做操作
- **元素编辑**：调整大小、旋转和移动元素
- **样式定制**：自定义颜色、边框和其他样式属性
